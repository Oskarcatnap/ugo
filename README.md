# 🐾 UGO - Cat Labyrinth

**UGO** — это минималистичный и хардкорный пазл-лабиринт, написанный на Python с использованием Pygame-CE. Пройдите через 50 уровней, маневрируя между стенами, чтобы достичь цели.

## 🎮 Особенности
* **50 уникальных уровней:** От очень простых до реально сложных.
* **Адаптивное окно:** Размер экрана подстраивается под каждый уровень (никаких черных полос!).
* **Senior Code:** Чистая объектно-ориентированная архитектура.
* **Легкий вес:** Игра запускается мгновенно.

## 🕹 Управление
* **Стрелочки** — передвижение котика.
* **R** — перезапустить текущий уровень (если застрял).
* **ESC** — выход из игры.
